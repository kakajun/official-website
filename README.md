# official-website

深圳市汇择科技有限公司官网, 用 Bootstrap v5.3.1 改造而成

## 预览

[官方 Demo](https://kakajun.github.io/official-website/)

## 特点

1. 响应式布局
2. 手机端适配
3. 兼容性良好
4. 不必打包
5. 轻量级
6. 持续更新

## 快速开始

1. 下载
2. 开发时在dev.html 中修改内容, 发布时压缩css 和html, 可以跑minify.bash
3. 运行 index.html
